## FORMAT DES FICHIERS AUDIO

- Format : **MP3** (recommandé) ou M4A
- Qualité : 128 kbps minimum
- Uploader dans : **Supabase Storage** → bucket `jardin-quarterons-media`
- Copier l'URL publique → coller dans le champ `audio_url` correspondant

## FORMAT DES IMAGES

- Format : **JPG** ou **WebP** (recommandé)
- Images circulaires : ratio **1:1** (carré), minimum 400×400px
- Image 360° : ratio **2:1** (panoramique), minimum 4000×2000px
- Uploader dans : **Supabase Storage** → bucket `jardin-quarterons-media`
